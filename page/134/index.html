Fast Android Networking

分类于 Android Valine：


Fast-Android-Networking

Difference over other Networking Library

  1. In Fast Android Networking Library, OkHttpClient can be customized for every request easily.
  2. As Fast Android Networking Library uses OkHttp and Okio, it is faster.
  3. Single library for all type of networking.
  4. Supports RxJava, RxJava2 -> Check here
  5. Current bandwidth and connection quality can be obtained to decide logic of code.
  6. Executor can be passed to any request to get response in another thread.
  7. Complete analytics of any request can be obtained.
  8. All types of customization is possible.
  9. Immediate Request is really immediate now.
  10. Prefetching of any request can be done so that it gives instant data when required from cache.
  11. Proper cancellation of request.
  12. Do not cancel a request if completed more than a threshold percentage.
  13. Simple interface to make any type of request.
  14. Proper Response Caching, hence reducing bandwidth usage.
