—— OPPO 高级手机产品经理施福来

Find 系列手机在国产手机市场里算是一个异类，这不光体现在它一贯拔尖的售价，也体现在它丝毫不在意一年一度一期一会地发布节奏，不是产品就时间，而是时间随产品。


▲ Find X

最异类还是属于它产品层面，几乎每一代都有独到之处，这种独特感在 Find X 上表现得更为明显：前后无孔加升降结构，让这款手机的颜值在 2 年后依旧出众，更不要说当时的惊艳感了。

这种对设计的期待感自然也随之顺延到了 OPOO Find X2 系列上，不过有些让人意外的是，Find X2 系列的设计并不先于时代，素皮以及陶瓷材质的加入，以及曲面屏的观感，还有刷新记录的「下巴」宽度等等固然优秀，但还没到前代让人 Wow 一下的水平。

所以，疑问就来了。

为什么这一次 Find X2 双机没有沿袭 Find X 未来主义的设计形态，变得更加的实用主义倾向？

在发布会后的采访里，OPPO 高级手机产品经理施福来做出了解释：

我们发现 Find X 防水防尘重量等层面的需求上得不到满足。所以我们综合考虑设计、防水、影像、5G 以及续航，在这一代放弃了升降结构，采用曲面的挖孔全面屏设计。

对于手机产品经理来说，平衡设计和功能是一件极其困难的事情，消费者需要超强的性能，但高性能芯片会带来续航和散热的压力；消费者也需要超长续航，但长续航要大电池，大电池就得更重更厚…

再比如，在屏下摄像头没法量产的当下，用了升降结构就要放弃防水防尘，还得增加重量浪费机身空间；用挖孔屏那么就无法实现正面绝对纯净的观感。而前后无孔的设计也压缩了手机摄像头的空间，无法和更大面积 CMOS 主摄以及多倍变焦长焦镜头共存。

权衡之下，这次发布的 Find X2 Pro 上，天平明显倾向到了功能这边，身兼 DxO 并列第一的相机和 DisplayMate A+ 的屏幕，再加上骁龙 865+X55 的性能网络，还有 LPDDR5 内存都是业界最佳级别的。

当然，这并不意味着 Find X2 Pro 在功能上就是尽善尽美的，比如在采访中也有人问，为什么没有配上 30W 的无线充电功能，毕竟这个功能已经在去年年末发布过。

这里自然也是权衡的结果，如果要加入无线充电功能，那么就得在手机背板上加入感应线圈，加了线圈那厚度和重量也得略微增加。

对此，施福来说：

当时我们在定义产品的时候纠结了很久，到底要不要加无线充电的技术？其实我们也看到智能手机要保证续航，特别是在 5G 时代，整机越做越厚。我们如果加上一个无线充电模块，整体可能要增加 0.3mm 左右。

我们综合权衡，因为我们有比较好的 SuperVOOC 技术，我们比较自信地把无线充电的技术先不在 Find X2 采用，从而保证整机手感。

前面看到 Find2X 标准版整机只有 8cm 的厚度，重量大概是 187g 的样子，整个手感做到非常棒的状态，主要是取舍与平衡的问题。

OPPO Find X2 Pro 深度影像评测

一个合格的手机摄影评测至少应该满足三点：

  • 足够丰富题材支撑的充足拍摄体量。
  • 拍摄者有着自己的审美倾向和摄影理念。
  • 能够传递实际的拍摄体验。

一定要花很长的时间和精力进行实际的拍摄积累。

值得骄傲的三摄镜头
很强，是真的很强。而且如果从画质上来说，可以用无短板来形容。自从多颗摄像头以来，摄像头就有了主次之分。

也正是这种主次之分，让消费者已经习惯了不同镜头之间的成像差异。比如在多镜头的初期，第二颗镜头一般都是长焦镜头。而长焦镜头的画质和主镜头可以说是天壤之别，尤其是在暗光环境下。

但在 OPPO Find X2 Pro 上，我认为已经基本上没有所谓的主次镜头之分，因为每一颗镜头都已经做到了很高的素质。而实现这一点最重要的原因就是「堆料」的硬件基础，而且这种堆料甚至是有一种降维打击的意味。

超广角直接使用了 IMX586 。听不懂没关系，你只要知道这款传感器在别的手机上一般都是作为主镜头的底在使用就可以了，这就是我说的所谓的降维打击。

其实广角摄像头的画质这几年来基本上都是被「牺牲」的部分，因为超广角镜头的特殊性在于夸张的视角，所以厂商基本上都是满足视角的新奇以后忽略了画质，这一点即使目前的 iPhone 11 Pro 系列都是如此。超广角的素质在光线充足的情况下尚可，但是在弱光环境下超广角素质下降十分明显。

但 OPPO Find X2 Pro 的超广角即使在弱光环境中，也表现尚佳。这大大提高了超广角的应用场景。尤其是一些室内场景，还能够做到相对纯净的画面，这在今天的智能机是对绝对的少见。光线充足的情况下，我们认为基本上超广角的画质和主镜头画质没有明显的差异。这颗镜头的基本素质很高，包括像是畸变和色散都控制在一个可以接受的范围。

坦率地说，我一直认为手机的超广角是最难驾驭的一颗镜头，别说大部分普通用户，即使是专业的摄影师使用超广角都是「技术活」。超广角就像是一匹野马，用的好了是「自由自在」，用的不好了是「鼻青脸肿」。想要让消费者爱上这匹野马，前提是可以让消费者随时拉出来溜溜，而不是每次使用之前都会担心画质。

潜望式镜头已经不再是什么黑科技，而且你需要知道的是，潜望镜头宣称的倍数都不做数，比如 OPPO 宣称最高可以 60×，但是你实际拍摄时，基本上永远也不会用 60×。

从我们的实际使用体验来看，5× 画质属于可用，10X 画质就是一个临界点。要注意的是，我们所说的 5× 已经包含了弱光环境。虽然 5× 和 60× 差距不少，但是实际上 5× 变焦画面的大小差别已经非常巨大，比如下面这一系列变焦画面。

自从我拍摄了第一组变焦作品以后，我就喜欢上了 OPPO Find X2 Pro 的长焦镜头。对于普通用户来说，超广角是山珍海味，虽然第一次吃的时候无比新鲜，但是不是家常饭。

但是长焦不一样，长焦虽然视角上平平无奇，但这才是平平淡淡的柴米油盐。可惜的是，因为长焦镜头本身物理光学的限制，所以一直没有办法在智能机上有一个跨越。但是自从潜望式镜头的出现，这件事终于出现了转机。

主流的智能机主镜头等效焦距基本上是 26mm，这个焦距其实在相机领域看来依然是属于广角范围。但广角只是摄影镜头的三分之一，镜头焦段大致分起来还有 50mm 左右的标准焦距，100mm 以上长焦距。

因此在很长一段时间里，手机摄影的硬件都是有「焦段空白」。这件事直到多镜头智能机的普及，50mm 才进入用户的视野。但虽然有了，与超广角一样，标准焦距画质也有着明显的差距。越长的焦距，就需要更多的光线，这就是问题所在。

但OPPO Find X2 Pro 不仅在 50mm 左右标准焦距（2×）上有着很好的表现，在 100mm 左右的长焦端（5×）也已经到了可用的程度。

其实摄影师追求变焦，放大缩小只是一部分。在我看来长焦镜头的真正意义在于「简化」和「压缩」。长焦可以让我们更加轻松去掉那些画面中不需要的部分，更好的去做摄影的减法。而「压缩」正好是和超广角相对，超广角镜头之所以夸张，是因为近大远小的透视关系被放大，而长焦则完全相反，把空间内的物体之间相互的距离模糊。这种压缩，在拍摄远处的建筑时，会有明显的感知。

虽然这听起来会有一些标题党，但是 OPPO Find X2 Pro 的变焦画质已经有一种「变焦卡片相机」的味道。虽然画质上短时间还有很多追赶的空间，但是对于大多数人来说，已经是一台很好的记录机器。更关键的还是那句话，再好的卡片机永远随时都在你的身边，但是手机可以。

在超广角和长焦的优秀表现下，倒是显的主镜头平平无奇，但其实主镜头更进一步。

既然别家主摄镜头的传感器都已经被用作超广角镜头，那么自己的主镜头自然要更好一些。所以 OPPO 是用了一块直接定制的 4800 像素的 1/1.4 英寸的传感器。卡片机也不过是 1 英寸而已。

耐人寻味的是，4800 万像素在今天的智能机上已经不再是什么新鲜事。毕竟一亿像素已经成为了热门话题。但越是专业的摄影师越是会在相机的像素上「冷静」。

爱恨纠葛的屏幕
我本身是很喜欢这块屏幕的。其实我这里本来可以罗列一堆参数，其中包括什么 120Hz 的超高刷新率、513 PPI 像素密度、10bit 色彩等等。但这些参数过于干瘪，其中很多参数，如果你无法上手，根本无法获得非常明确的感知。

所以在拿到手机之前，我们还真的有很多的期待。因为高素质的屏幕很长一段时间真的只有在苹果和三星的顶级旗舰上可以见到。尤其是色准，这一直是很多国内手机厂商忽略的部分。

事实是，OPPO Find X2 Pro 做到了。它是我们在安卓手机上看到的最好的屏幕。无论你用什么样的形容词来赞美都不过分。

但你可能更想问的是，屏幕和摄影有关系吗？影像的评测为什么要提到屏幕？当然有关系，而且有着非常重要的关系。

任何一个有经验的手机摄影师，都清楚屏幕的重要性。身边的很多手机摄影师，因为工作的原因可能会同时使用 iPhone 和安卓，基本上安卓拍摄照片都会传递到 iPhone 上进行后期。

原因也非常简单，安卓机无论是从硬件、软件上在色彩表现和色彩管理上都不如 iPhone。使用这样的屏幕后期，无异于闭眼后期，辛辛苦苦完成的作品发布到社交网络不是偏色就是过饱和。

手机摄影后期是手机摄影重要的组成部分，甚至从我手机摄影的经验来看，摄影后期可以是手机摄影最有趣的一部分。简单的到滤镜的使用，复杂的到创意后期的变化，都需要依靠一块优秀的屏幕来完成。

所以，一块优秀的屏幕是手机摄影的重要基石。

摄影后期只是屏幕最显性的一部分，还有很多隐性影响。比如你的屏幕素质优秀，那么在预览和回放照片时就更能让摄影师感受到画面中的细节精彩之处，这可以很快的给予摄影师一个正向反馈。这些正向反馈可以激励摄影师更好的进行创作，没有任何一个摄影师不需要好作品的鼓励。

这一点在我们拍摄彩色流体的作品时感触最深。当我们把彩色的颜料倾倒在在画布上，看到色彩开始流动时，OPPO Find X2 Pro 很好的传递了这种颜色视觉感受。

这让我们对于这组作品的创作欲望达到了一个很高的点，所以我们开始在颜料的不同部分不断地拍摄，不断地寻找，再不断地拍摄。两个人甚至会不断地和对方分享自己拍摄的精彩画面。

拍摄色彩流动的视频时这种拍摄兴奋度到了顶点，因为曲面屏的溢出效果是一种奇妙的视觉感受。不过说到曲面屏，只可惜，除了兴奋还是难受多一点。

从实际的拍摄体验来看，曲面屏对于手机摄影还是帮倒忙多一些。这也是为什么在这一节，我使用了爱恨纠葛这个词，恨就是来自于这里的曲面。

曲面最大的一个问题就是视野的不完整。摄影构图不仅仅是为了美学追求，更多的是一种杂质过滤。这种过滤最直观的方式就是通过界定边界。

也就是说，一个摄影师需要清楚的知道哪里属于作品内，哪里属于作品外。曲面屏恰恰就是模糊了这种界限。这让摄影师很难把控边缘的细节。所以我们在拍摄时，经常需要特意检查每一个边界，来判断是不是自己想要的画面。

这无形之中，就添加了一种负担。

我不确定是不是高刷新率带来的决定性改变，但是一定是原因之一，这就是 OPPO Find X2 Pro 的相册预览体验很好。打开相册以后，一次性可以完整展示 32 张作品，配合着细腻的显示效果，让自己的作品更耐看。最让我意外的是，整个相册滑动起来的体验极佳，以往安卓手机相册预览的卡顿感完全没有，这种顺滑的体验以前可绝对是 iPhone 专属。

总的来说，OPPO Find X2 Pro 这块屏幕会给人留下非常深的印象，它是 OPPO Find X2 Pro 影像拍摄体验的重要支撑。

不是噱头的双原生 ISO

这又是一次手机摄影和相机摄影的交汇点。因为双原生 ISO 在专业相机上不罕见。但我们先放下这些概念不谈，先直接来说说 OPPO Find X2 Pro 的夜景表现。

优点很直接，OPPO Find X2 Pro 的夜景模式所有镜头都可以使用，也就是说无论是超广角、广角、长焦镜头都可以夜景模式。主镜头的夜景模式已经是智能手机的标配，但是超广角和长焦的夜景模式则经常有缺憾。

其实超广角的夜景模式很重要，因为城市大风光夜景最常见的焦段就是超广角，所以如果超广角能够配合夜景模式使用，就可以同时获得画面视觉效果震撼、画质细节优秀的夜景佳作。

整体上，OPPO Find X2 Pro 的夜景模式曝光倾向还是倾向于明亮。好处就是拍摄出来的照片整体观感很好。尤其是在一些城市夜景上给人以「星光璀璨」的感觉。而夜景模式的色彩也非常出众，无论是远处的蓝色还是近处的车光暖色在还原的基础上还有适当优化。

坏处就是很容易出现部分亮部过曝，好在范围可控，没有出现大面积的过度曝光情况。

其实风光摄影师有一个重要的道具，就是三脚架。三脚架的意义在于可以拥有更长的快门时间，从而可以用更低的 ISO 即感光值。而更低的 ISO 就意味着更加纯净的画面表现。这也是数码相机光学传感器的一个重要特征。

所以通常来说，一个传感器总会有一个最佳的画质。而所谓的双原生传感器就是拥有两个最佳画质的感光值，这需要在「硬件层面」上支持，也就是说这并不是单纯的通过软件调校就可以获得的结果。

OPPO Find X2 Pro 的双原生 ISO 分别是 ISO 100、ISO 800，很明显前者对应的强光环境，而后者对应的是弱光环境。很多人在使用手机拍摄的时候，都会发现一个问题，这就是夜晚的画质下降是断崖式的，涂抹非常严重。

这就是因为相机此时为了曝光量只好在 ISO 100上放大到 800 或者更高感光值，所以会出现噪点等等影响画质的问题。但如果我们有了原生 ISO 800，就意味着我们在夜晚拍摄时调整感光值在 ISO 800，没有放大过程，因此会更加纯净。

如果你的 OPPO Find X2 Pro 是一个胶片相机，就意味着它有两个不同感光度的胶卷可以选，你可以根据需求切换。

当然，这都是理论上的。想要了解原生感光值 800 是不是真正起到了作用，只要拍摄一张感光值为 ISO 700（或者更低）的照片和 ISO 800 进行对比纯净度即可：

很明显，ISO 800 更纯净一些。

所以，如果你有机会使用 OPPO Find X2 Pro 拍摄星空或者是任何长曝光作品，使用 IOS 800 会有着更好的画面表现。能够有足够的星空亮度，还能保持画面的纯净。只可惜，因为疫情被困在上海在这里无法展示样片。

双原生感光值对于的夜景表现是体现在每一张夜景照片中，它是 OPPO Find X2 Pro 夜景纯净的重要支撑。

真正的微距
大多数手机厂商的微距都是为了在发布会上的 PPT 占地方，但是 OPPO Find X2 Pro 不是。

微距镜头的奇妙如果你没有真正使用过，很难有所体会。就算我最后把拍摄好的作品给你看，这种新奇感也会降低。

我自己本身很喜欢微距镜头，所以在相机的镜头群里也会有一只微距头。微距可以让很多看起来非常稀松平常的画面变得不普通，比如下面这一张照片：

其实拍摄的地方不过是南京东路旁边再普通不过的一个小花坛，但是因为微距的存在，就可以获得完全不一样的视角。

其实原本我们的彩色流体拍摄是想来测试 OPPO Find X2 Pro 的屏幕色彩素质，没想到我们调配好颜色以后，无意中发现 OPPO Find X2 Pro 在靠近物体拍摄时会自动切换为一个微距模式。

而这种微距模式可以非常细腻的展示出流体的纹理，所以我们就开始不断地在画布的不同部分去发现这些细节和纹理。

其实微距镜头不是第一次出现在智能机上，但是 OPPO Find X2 Pro 的微距镜头是我接触过的真正值得称得上「微距」的。

首先它有着很好的放大倍率，真正可以做到放大细微之处。其次微距拍摄的画质极佳，无论是色彩还是细节都给人留下了深刻的印象。

最有趣的点在于，如果不用微距模式色彩还原反而会差一些。而如果使用微距模式，色彩反而真实和饱满。这的确让我们感受到了一头雾水。不过我们猜测大概率是因为在手机识别微距以后会从主镜头切换到超广角镜头拍摄进行裁切的原因。

但原理不重要，结果就是 OPPO Find X2 Pro 的微距很值得用。

提到防抖，的确是 Pro 的视频
对于入门的视频制作者有两大难题：一个是声音，一个就是视频的稳定。而恰恰这两点又非常影响影片的质量，极差的声音和晃动的镜头都会给观看者非常严重的生理不适。

所以这也是为什么这几年以来，所有的智能机都在关注视频防抖这件事，想要解决这个问题有两个出发思路。

一个是光学防抖让传感器动起来来弥补手持过程中的晃动。这种防抖效果最真实，但是因为传感器本身的位置受限，所以防抖效果也就受限。

第二个出发思路是算法，简单来说就是先把所有的镜头画面找到一个晃动的合适范围，然后进行画面的裁切再对齐。这样的方式效果最明显，可这会牺牲一定的画面面积。所以理论上来说，一旦开启防抖画面范围就会变小。

在了解到以上基础以后，我们先说结论：

OPPO Find X2 Pro 的防抖很好，对于绝大多数普通用户完全可以替代手机稳定器。

这听起来很夸张，但是只要你一上手机就可以立刻感受。当然，这里的替代仅仅是防抖层面，而手机稳定器的作用不仅仅是防抖。

但更有意思的是，我们发现开启最好的 Pro 防抖以后，效果明显更好的同时，画面内容竟然比未开启 Pro 防抖的还要多。这乍看起来和我们前面说到的原理是相违背的。但其实经过我们几次尝试以后就发现了其中的「奥秘」：

「偷梁换柱」

我们发现 OPPO 的 Pro 防抖开启时不再使用主要的广角镜头拍摄，而是使用超广角镜头。这样在防抖裁切以后便可以获得更大的拍摄画面。可其实想要这样做，有一个重要的前提，这就是超广角镜头本身画质要够硬。而这颗超广角镜头的画质够不够硬答案前面显然已经说过了。

有一说一，这一点是我认为 OPPO Find X2 Pro 最值得赞赏的思考。单纯的堆硬件不是关键，关键的是如何充分利用。既然我的超广角画质更好，就尝试思考还能做什么。这一点和今年的 iPhone 在使用了超广角以后提出了超取景框拍摄的概念有着异曲同工之妙。

除了视频的防抖以外，这一次 OPPO Find X2 Pro 的视频另一个核心特点是 HDR 。官方叫做 Live HDR，HDR 技术已经不再是什么新鲜事。想要测试 HDR 效果的好坏很简单，只要找到一个光影对比强烈的场景即可。所以我们在中午十二点左右选了一些光影对比强烈的画面进行了拍摄，下面是一些视频画面的截图：

基本上，开启 Live HDR 以后整体的画面素质在阴影亮度和高光压制上都有着很好的表现。尤其是在一些光比明显的环境中，对于人物肤质的还原表现不错，这充足了保证了模特脸部画质。

整体来看，OPPO Find X2 Pro 在视频保持曝光的均衡上可以说可圈可点，基本上很少出现曝光抽动的情况。但是色温就没有这么幸运了。我们在拍摄色彩流体的部分中，用视频进行了记录。考虑到视频拍摄画面的一致性很重要，所以拍摄固定机位锁定曝光和对焦已经是常规操作。可即使如此，OPPO Find X2 Pro 的色温还是出现了「抽动的现象」。我们可以明显从下面的两帧画面之间看到区别：

所以，这也是我们强烈希望在后续 OPPO 的工程师能够在锁定曝光和对焦的时候锁定色温。不过瑕不掩瑜，如果你看中视频的防抖，OPPO Find X2 Pro 是真的强。

稍有叛逆的对焦
其实对于手机用户来说，对焦是一个很难感知的部分。智能机的景深由于传感器面积、焦段的因素等等都非常深。所以很多时候即使对焦并不精准也不会有很大的影响。但随着焦距越来越长，光圈越来越大，手机对焦这件事也应该被重视。

官方宣称OPPO是全像素全像对焦，这听起来就很「全」。

但是 OPPO Find X2 Pro 的实际对焦体验却有点「叛逆」。首先在拍摄一些抓拍场景中，OPPO Find X2 Pro 的表现可以说是绝对的合格。例如下面这一张照片：

拍摄场地附近有一些巨型的橙色钢铁建筑，可以说是非常好的超广角背景。所以我决定躺在地面向上拍摄，用来展现超广角的夸张视角。而为了画面更加灵动，所以我决定让 Jamie 从我的头顶飞过，抓拍跳跃的瞬间。

这听起来没什么难的，但是这样的拍摄在很长一段时间里我都很少在安卓手机上尝试。因为想要能够抓到最好的瞬间，有两个非常重要的基础：

快门的延迟一定要低。因为我只能通过 Jamie 进入画面的一瞬间就按下快门。如果快门延迟高，那么拍摄时 Jamie 很有可能已经飞过去了；
对焦速度不仅快，还要准。只有这样才能获得更加清晰的照片。
本以为需要多次尝试的我们，第一次就获得最满意的一张照片。很明显，OPPO Find X2 Pro 的抓拍对焦的确很好。不仅仅是广角，在长焦段「打鸟」都很迅速。

但超广角和长焦镜头还没用到全像素全向对焦，配备了该能力的主摄镜头抓拍更是精准迅速，尤其是在宠物摄影上有明显感知。

但有意思的来了，如果让 OPPO Find X2 Pro 自己抓绝对是稳准狠，但是如果你要是想要告诉拍什么，它开始「不听话了」，这是我和 Jamie 都会有的感受。

在一些场景中，需要我们自己选择画面拍摄的焦点。一般来说，智能机最直接的方式就是点按画面，但是只要我们一点按画面就会出现一定几率的对焦失败。这种情况弱光居多，但是强光也有。有的时候，即使点按了几次还是会出现对焦失败。

这大概率应该是软件调校的问题，所以希望后面的软件更新会有所改善。

喜忧参半的操控
其实从我的拍摄经验来看，智能机的画质并不是最高优先级，操控体验才是。但长期以来，安卓手机的相机操控体验实在是称不上优秀。

坦率地说，OPPO Find X2 Pro 的操控逻辑还是经典的「安卓味道」。整个相机给人的印象依然是冗杂。该简洁的地方没有做到简洁，该给更多的反而没有给更多。

一个小细节，比如时时刻刻都在提醒我美颜的「滤镜强度选择」。我并不是对于美颜这件事有意见，但是对于手机摄影来说，美颜真的不至于在右下角直接配备一个单独的图标。哪怕这个图标直接排列在拍摄界面上的正上方也好，直接内嵌在画面中，实在是过于引人瞩目，彷佛是时时刻刻提醒我，你「需要美颜」一下。

该详细的地方在于，我一直很奇怪为什么很多安卓手机里有「更多」这个选项。慢动作、延时摄影真的值得在第一层级出现。而不是每一次都需要单独的进行一次点击。很长一段时间里，慢动作和延时摄影是安卓手机的短板，所以这种「藏起来」的方式我可以理解。但是今天安卓手机的视频规格至少在硬件上已经是平起平坐了，应该大方的拿出来。比如 OPPO Find X2 Pro 下面这一段慢动作：

我认为以上两点的问题在于，OPPO Find X2 Pro 需要更坚信自己专业摄影手机的定位。

不过话说回来，安卓机自带「专业模式」绝对值得一夸。OPPO Find X2 Pro 当然也具备了专业模式，这给了摄影师更多的选择和创作空间。

首先最喜欢的是镜头的选择。因为在正常的拍摄界面变焦是数码和光学混合的，很多时候你以为的长焦不过可能是一种裁切。但是当我想要最好的画质时，肯定是用对应的光学焦段，此时你就可以通过专业模式在三颗镜头之间切换，有一种单反换镜头的感觉。

其次是参数的自主控制，比如想要拍摄城市车轨，可以通过降低 ISO，延长快门速度获得。对了，还有很重要的色温，固定的色温可以保证一系列作品的颜色统一。

但我最开心的是 OPPO Find X2 Pro 终于开始关注到了一些细节的「拍摄反馈」。比如，我们在转动变焦拨盘时配合了优秀的震动，同理也包括在专业模式的参数调节时。这一点在我上手拍摄的第一时间就有多感受。这很快就让我喜欢上了滑动变焦拨盘，而不是所谓的点击变焦。

这其实真的非常重要，在相机上，机械的震动可以明确的给摄影师反馈，但是智能机的触摸恰恰去掉了这个过程。所以如果可以通过优秀的振动体验进行还原，能够让拍摄的过程更有参与感。

你该知道的细节问题
连拍只有 20 张
虽然连拍有 20 张看起来已经很多，但是实际上 20 张的拍摄时间很短。每次使用到连怕，基本上都是为了能够捕捉到一些高速的瞬间，所以往往是需要我们提前预判拍摄。但是我们在使用 OPPO Find X2 Pro 很多次都遇到了最好的动作还没出现，就已经到了抓拍的数量限制。

出现了一次蓝屏 Bug。在一次光源充足的拍摄中，突然连续多张出现了蓝色条纹的照片。但是仅仅是出现了一个拍摄场景，也仅仅出现在我的这部手机上。虽然这可能是大概率的个案，但是因为真实出现了，所以我还是决定写下来。

视频拍摄没有网格线
这也是一个后期可以修复的 Bug。我一直说，看一个人是不是认真看待手机摄影就看他拿到手机的第一件事是不是先打开手机的网格。

所以我和 Jamie 在拿到 OPPO Find X2 Pro 以后第一件事就是打开网格线。但是在视频拍摄中，即使我们打开了网格线，它依然没有出现。

曝光调节杆高误触率
最后这个问题是整个拍摄过程中，唯一一个一度让我生气的点。因为想要更好的画质和光线氛围，在使用手机拍摄过程中就需要对手机进行曝光的调整。

一般方式是通过点击画面，然后滑动对焦框旁边的滑杆来实现，但是 OPPO Find X2 Pro 经常出现一滑动滑杆就会切换拍摄模式的情况。有一次拍摄甚至连续多次滑动都会出现这种误触，导致手机莫名其妙就切换到「隔壁」，比如照片到人像。

专业模式曝光无法预览
专业模式可以让摄影师为了特定画面效果调整参数，但此时 OPPO Find X2 Pro 预览界面不能很好地给予一个反馈，所以很多时候需要反复盲测。

尾声
好了，到了这里，也是时候回答文章开始的两个问题：

OPPO Find X2 Pro 的影像系统值得这个价位吗？

答案是非常肯定的。在这个价位上比拼手机摄影的机器，一个大前提是要能有着非常重要的基本功。这里的基本功是指你每一次按下快门要有很快的响应速度、果断的对焦体验、稳定的白平衡表现和上等的成片质量。而以上这几点，OPPO Find X2 Pro 虽然称不上完美，但是实力绝对属于第一梯队

但是有基本功还远远不够，iPhone 摄影之所以能有如此高的地位还有其本身的「独特个性」。而我们欣喜地发现，这种个性我们在 OPPO Find X2 Pro 上也发现了。

非常扎实的三摄系统，已经不仅仅是可以用简单的堆料来形容而已而是从硬件到软件都做好了准备。

顶级的屏幕素质，不仅仅是体现在最后成片的预览上，而是在整个摄影体验流程中都能相得益彰。

而也正是这两点，让 OPPO Find X2 Pro 非常有「记忆点」。这让我们在拿起其它手机拍照时，会有那么几个瞬间会想，如果我现在用的是 OPPO Find X2 Pro 就好了。

但这是一个开始，而不是结束。它依然有很多值得改进的空间，这其中有：

短时间内可以通过软件修复的 Bug，比如消失的视频参考线；
也有需要长期以来摄影审美积累的沉淀，比如对于所谓 AI 的思考；
更有安卓生态本身带来的多年顽疾，比如糟糕的摄影后期生态。

那么，这就涉及到我们想说的第二个问题，安卓手机摄影。

长期以来，安卓手机在影像实力上一直给人以「理工」出身的印象。每一次发布都通过压到的参数来让消费者相信自己的影像能力。

但手机摄影的本质依然是摄影，它在内核上依然需要走完一次非常完整的「摄影表达过程」。从摄影师点亮屏幕开始取景的那一刻起，每一次和手机的互动都会成为手机摄影的重要一部分。

很显然，OPPO Find X2 Pro 已经开始从「点亮屏幕」开始了。
